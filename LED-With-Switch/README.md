# LED With Switch

## Description
Implemented a push-button controlled LED using the AT89C51 (8051) microcontroller. The LED turns ON or OFF based on the switch input.

## Components Used
- AT89C51
- LED
- Push Button
  

## Software
- Keil uVision
- Proteus

## Concepts Used
- Digital Input
- Digital Output
- Port Interfacing
- Embedded C
