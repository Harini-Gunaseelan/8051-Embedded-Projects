# Single 7 Segment Display

## Description
Implemented a common cathode 7-segment display using the AT89C51 (8051) microcontroller. The display counts from 9 to 0 using Embedded C.
Demo link : https://youtu.be/hpTMp_fVkmk

## Components Used
- AT89C51
- Common Cathode 7-Segment Display
  

## Software
- Keil uVision
- Proteus

## Concepts Used
- Array Lookup Table
- Port Interfacing
- For Loops
- Embedded C

## Output
Displays digits from 9 down to 0 repeatedly.
