# Four-Way Traffic Light Controller

## Description

Implemented a four-way traffic light control system using the AT89C51 (8051) microcontroller. The system controls traffic flow at a four-road junction by sequentially activating traffic signals for each direction.

## Components Used

- AT89C51 Microcontroller
- LEDs (Red, Yellow, Green)
  

## Software

- Keil uVision
- Proteus

## Concepts Used

- GPIO Port Interfacing
- Traffic Signal Sequencing
- Timing Delays
- Functions
- Embedded C

## Working Principle

The controller activates traffic signals for one road at a time while keeping the remaining roads in the stop state. After a predefined delay, control is transferred to the next road, ensuring orderly traffic movement through the junction.

## Features

- Four-road traffic control
- Sequential signal operation
- Configurable timing intervals
- Simple state-based control logic
