# Common Anode 7 Segment Display

## Description
Implemented a common anode 7-segment display using the AT89C51 (8051) microcontroller. The display counts from 9 to 0 using Embedded C.
Demo link : https://youtu.be/hpTMp_fVkmk

## Description
